# edoctor-ai

Voice AI Assistant.

## Prerequisites

- Node.js >= 22 (LTS)
- Deepgram Account
- Groq Account
- Azure Account

## Setup

> Change the terminal to `Bash`

Install packages:

```bash
npm install
```

Run:

```bash
npm run dev
```

## Note

Default language is Vietnamese. You can switch to any supported language in the Deepgram settings
